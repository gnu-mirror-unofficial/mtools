Mailing list and documentation

There is an mtools mailing list at mtools@mtools.linux.lu . Please send all bug reports to this list. Note: You must be subscribed to the list in order to be able to send comments. You may subscribe to the list by visiting:

https://mtools.linux.lu/mailman/listinfo/mtools

Announcements of new mtools versions will also be sent to the list, in addition to the linux announce newsgroups. There also exists an archive of the mailing list (currently only accessible to subscribers).

A HTML version of the manual for mtools is available here, and a PDF version is here This manual will only be updated for major releases, or when there is a significant change in it.
